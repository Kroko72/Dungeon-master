Для работы игры необходимо:
1) Создать папку "data" (название без кавычек) и перекинуть туда все картинки и звуки,
идущие вместе с программой.
2) Установить необходимые библиотеки из файла requitements.txt

Управление:
Управление персонажем происходит на клавиши WASD, которые отвечают за передвижение вверх,
влево, вниз и вправо соответсвенно. Для выстрела необходимо нажать на левую кнопку мыши,
а для возвращения стрелы назад нужно зажимать правую кнопку мыши (когда стрела уже выпу-
щена). Для переката нужно зажать во время движения пробел и персонаж сделает перекат, во
время которого он двигается быстрее, в сторону соответствующую зажатым клавишам. Чтобы
приостановить игру нужно нажать клавишу ESC (эскейп).
